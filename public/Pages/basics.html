SQL Server Interview Guide - Basics

SQL Server Basics - Questions and Answers

1

What is SQL Server?

SQL Server is a relational database management system (RDBMS) developed by Microsoft. It is used to store, manage, and retrieve data for various applications.

2

Explain the concept of Primary Key in SQL Server.

A primary key is a column or a set of columns that uniquely identifies each row in a table. A primary key must contain unique values, and it cannot contain NULLs.

CREATE TABLE Students (
    ID INT PRIMARY KEY,
    Name VARCHAR(50)
);
3

What is the difference between a clustered and a non-clustered index?

A clustered index determines the physical order of data in a table. Only one clustered index can be created per table. Non-clustered indexes are separate structures from the table and can be created multiple times.

4

What is the purpose of the SELECT statement in SQL?

The SELECT statement is used to query the database and retrieve data from one or more tables.

SELECT * FROM Students;
5

What is normalization?

Normalization is the process of organizing data to reduce redundancy and improve data integrity. It involves dividing a database into two or more tables and defining relationships between the tables.

6

What is a FOREIGN KEY constraint?

A FOREIGN KEY constraint is used to establish and enforce a link between the data in two tables. It ensures the consistency of data by restricting the values in the foreign key column to those present in the referenced table's primary key column.

CREATE TABLE Orders (
    OrderID INT PRIMARY KEY,
    CustomerID INT,
    FOREIGN KEY (CustomerID) REFERENCES Customers(CustomerID)
);
7

What is the difference between UNION and UNION ALL?

UNION returns distinct values while UNION ALL returns all values, including duplicates.

SELECT Name FROM Students
UNION
SELECT Name FROM Teachers;
SELECT Name FROM Students
UNION ALL
SELECT Name FROM Teachers;
8

What is a view in SQL Server?

A view is a virtual table based on the result-set of an SQL query. It can be used to simplify complex queries or to present data in a specific format.

CREATE VIEW StudentView AS
SELECT Name, Age FROM Students;
