SQL Server Interview Guide - Joins

SQL Server Joins - Questions and Answers

1

What is a JOIN in SQL Server?

A JOIN clause is used to combine rows from two or more tables based on a related column between them.

2

What is the difference between INNER JOIN and OUTER JOIN?

An INNER JOIN returns only the rows that have matching values in both tables. An OUTER JOIN returns all the rows from one table, and the matched rows from the other table. If there is no match, the result is NULL.

SELECT * FROM Orders INNER JOIN Customers ON Orders.CustomerID = Customers.CustomerID;
SELECT * FROM Orders LEFT OUTER JOIN Customers ON Orders.CustomerID = Customers.CustomerID;
3

What is a LEFT JOIN?

A LEFT JOIN (or LEFT OUTER JOIN) returns all records from the left table and the matched records from the right table. If there is no match, NULL values are returned for columns from the right table.

SELECT * FROM Employees LEFT JOIN Departments ON Employees.DeptID = Departments.DeptID;
4

What is a RIGHT JOIN?

A RIGHT JOIN (or RIGHT OUTER JOIN) returns all records from the right table and the matched records from the left table. If there is no match, NULL values are returned for columns from the left table.

SELECT * FROM Employees RIGHT JOIN Departments ON Employees.DeptID = Departments.DeptID;
5

What is a FULL OUTER JOIN?

A FULL OUTER JOIN returns all rows when there is a match in either the left or right table. If there is no match, NULL values are returned for the columns that do not have a match.

SELECT * FROM Employees FULL OUTER JOIN Departments ON Employees.DeptID = Departments.DeptID;
6

What is a CROSS JOIN?

A CROSS JOIN returns the Cartesian product of the two tables. It will combine all rows from the first table with all rows from the second table.

SELECT * FROM Products CROSS JOIN Suppliers;
7

What is a SELF JOIN?

A SELF JOIN is a regular join but the table is joined with itself.

SELECT A.Name AS Employee, B.Name AS Manager
FROM Employees A, Employees B
WHERE A.ManagerID = B.EmployeeID;
