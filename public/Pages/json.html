SQL Server Interview Guide - JSON Handling

JSON Handling in SQL Server - Questions and Answers

1

How does SQL Server handle JSON?

SQL Server provides functions to parse, read, and write JSON data. You can store JSON data in a string column and use built-in functions to query it.

2

What are the JSON functions available in SQL Server?

SQL Server provides several functions to work with JSON, such as JSON_VALUE, JSON_QUERY, and OPENJSON.

3

How do you extract a value from a JSON string in SQL Server?

You can use the JSON_VALUE function to extract a scalar value from a JSON string.

SELECT JSON_VALUE('{"name":"John", "age":30}', '$.name');
4

How do you return a JSON array in SQL Server?

You can use the JSON_QUERY function to return a JSON array or object.

SELECT JSON_QUERY('{"names":["John", "Jane", "Alice"]}', '$.names');
5

How do you insert JSON data into SQL Server?

You can insert JSON data into a column of type NVARCHAR.

INSERT INTO Users (UserData) VALUES ('{"name":"John", "age":30}');
